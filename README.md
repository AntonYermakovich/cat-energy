# Тестовое задание

## Цель задания

Сделать адаптивную, кросс-браузерную вёрстку лендинга по макету из
Figma. Вёрстка должна быть выполнена вручную, без использования
внешних CSS/JS библиотек (например, Bootstrap, Tailwind, React, jQuery и
т.п.). Допускается использование CSS-препроцессоров (например, SASS/
SCSS, LESS, Stylus) для упрощения написания стилей.
Результат — готовые файлы index.html, styles.css, и при необходимости
script.js, а также удалённый git-репозиторий с коммитами и README.
Задеплоенный проект на GitHub Pages будет большим плюсом.
